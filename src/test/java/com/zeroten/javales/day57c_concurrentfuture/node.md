#### 1.线程池面试题
   （1）使用线程池让线程协同工作？
        向线程池里提交3个任务，每一个任务都会返回一个执行结果？如何获取这3个任务的执行结果，并进行组合？
        1.1）A接口耗时：tA
        1.2）B接口耗时：tB
        1.3）C接口耗时：tC
        假如：tA>tB>tC
        串行：tA+tB+tC
        并行：tC
   （2）Runnable VS Callable
        1.1）Runnable是线程执行的接口（run方法）
        1.2）Callable只是线程池调用任务的封装，与线程无关
