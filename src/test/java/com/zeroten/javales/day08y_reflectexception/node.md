    （1）对象的创建，正常情况下，是使用空参构造；那么，直接使用Class来创建对象，
        即是调用对象的空参构造。
    （2）使用有参构造创建对象。

### 2.反射注解
    小结：
    1.反射作为补充内容，优先级很低,，极低
    2.注解，要求大家掌握使用方法，注解的反射，不需要大家深入挖掘
    
### 3.异常
    1.当发生异常之后，在异常之后的程序将不在执行。
    2.通过捕获异常，可以让程序继续执行下去。
    3.捕获异常，只能捕获指定异常。
    4.finally：无论是否发生异常，都一定会执行，优先级非常高。
    5.异常，无论如何最终一定要被处理掉。
