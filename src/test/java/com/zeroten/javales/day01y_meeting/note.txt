一、中台
    中台含义：中台是真正为前台而生的平台（可以是技术平台，业务能力甚至是组织机构），它存在的唯一目的就是更好的
             服务前台规模化创新，进而更好的响应服务引领用户，使企业真正做到自身能力与用户需求的持续对接。
    中台存在的目的：解决前台的创新问题。
    我们能正常理解的项目流程：页面 -> 服务器 -> 数据库
